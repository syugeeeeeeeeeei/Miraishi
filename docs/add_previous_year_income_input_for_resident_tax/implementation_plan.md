# 実装計画

## 1. スキーマ拡張

- `Deductions` に `previousYearIncome` を追加する。
- バリデーションスキーマに同フィールドを追加し、未設定時は0を許容する。

## 2. 入力UI追加

- シナリオワークスペースの「扶養・控除」セクションに前年度収入入力欄を追加する。
- 補助文言として「新卒等で前年度収入がない場合は0」を表示する。

## 3. 計算ロジック修正

- 住民税の計算ベースを「前年ベース」に変更する。
- 1年目: `previousYearIncome`
- 2年目以降: シミュレーションで算出した前年度の `taxableIncome`

## 4. 計算フロー表示修正

- 詳細ダイアログの住民税式を、住民税計算ベース（前年入力 or 前年課税所得）を使う表現に更新する。

## 5. テスト追加

- 1年目が入力値ベースで住民税計算されること。
- 2年目が1年目課税所得ベースで住民税計算されること。
