# 実装計画

1. `ScenarioWorkspace` にホイール切替ロジックを追加する。
- 切替判定関数を作り、入力フォーカス時は無効化する。
- 修飾キー（Ctrl/Meta/Alt/Shift）押下中は無効化する。

2. スクロール可能領域との競合を回避する。
- イベントターゲットから親方向へ走査し、縦スクロール可能な要素を検知する。
- その要素にスクロール余地がある場合はシナリオ切替を行わない。
- スクロール余地がない境界時のみ切替対象とする。

3. 切替感度を安定化する。
- ホイールデルタを蓄積して閾値超過時に1回だけ切替する。
- 連続切替を抑えるためクールダウンを導入する。

4. `onWheel` をカード表示領域に接続する。
- ScenarioCard を内包する表示領域に対してハンドラを登録する。

5. 検証
- `yarn typecheck`
- `yarn test`
- `yarn build`
