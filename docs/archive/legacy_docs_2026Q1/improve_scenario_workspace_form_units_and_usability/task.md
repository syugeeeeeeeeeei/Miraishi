# タスク

## 依頼内容
ScenarioWorkspace の入力UIについて、次の3点を改善する。

1. 金額入力フォームに通貨単位（円）を表示する。
2. 扶養・控除カードの「配偶者の有無」をトグルではなく「有 / 無」ボタン切替にする。
3. フォーム全体のレジェンド・ボタンなどを少し大きくして操作しやすくする。

## 追加条件
- 全画面表示時に、ワークスペース全体として破綻せず1画面内に収まる構成を維持する。

## 完了条件
- 金額入力欄で通貨単位が視認できる。
- 配偶者の有無がボタンで切り替えられる。
- フォーム要素が読み取りやすく、押しやすいサイズに調整されている。
- `yarn typecheck` / `yarn test` / `yarn build` が通る。
