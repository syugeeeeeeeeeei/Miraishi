# タスク

税金ルール（JSON）をアプリ上で編集可能にし、サイドパネル下部の歯車メニューからアクセスできる設定導線を実装する。

## 要求仕様

- サイドパネル最下部に歯車アイコンを配置。
- 歯車クリックで円形メニューを展開し、税金ルール・クレジット・インフォなど拡張可能なメニュー構成にする。
- 税金ルールメニューで中央フローティングダイアログを表示。
- ダイアログは誤操作防止のため、右上のバツかフッターの「変更せず閉じる」でのみ閉じる。
- ダイアログはフォームモードとJSONエディットモードをタブ切替可能。
- ダイアログ内編集は即時保存（ドラフト更新）だが、反映は「変更を確定」押下時。
- 確定時に税金ルールを適用し、計算を再実行する。
