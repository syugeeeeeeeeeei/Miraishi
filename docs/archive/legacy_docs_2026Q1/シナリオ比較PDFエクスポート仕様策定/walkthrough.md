## 今回の成果

- 「複数選択シナリオ比較PDFエクスポート」機能の仕様を確定した。
- UI導線、IPC、データ型、掲載項目、バリデーション、エラー方針を実装可能レベルで定義した。

## 重要な仕様決定

- 比較対象は `activeScenarioIds` を既定とする。
- エクスポートは 2件以上選択を必須とする。
- 年次範囲は 1〜50 年（指定年まで）とする。
- PDF生成は Electron 標準の `printToPDF` 方式を採用する。
- 1シナリオでも計算失敗した場合、レポート生成は失敗として扱う。

## 実装時の主な追加対象

- Main IPC: `export-scenario-comparison-pdf`
- Preload API: `exportScenarioComparisonPdf`
- UI: システムラジアルメニューのエクスポート項目と設定ダイアログ
- レポート描画: 比較用DTOを表示する専用レイアウト（印刷CSS込み）
