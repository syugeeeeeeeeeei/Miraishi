# タスク

## 背景
このリポジトリでは `electron-builder` を使用して配布物を生成している。  
利用理由としては以下が挙がっている。

- アイコン設定ができる
- 自動アップデートを設定できる

## 相談内容
`electron-builder` よりも、Electron アプリをより簡単にパッケージングできる選択肢があるかを確認し、現プロジェクトに合う方針を決める。

## この調査で見る観点
- 現在の実装で `electron-builder` に依存している範囲
- 現在の自動アップデート実装の実態
- 代替候補（特に Electron Forge）で同等要件を満たせるか
- 乗り換えコストと運用コスト
