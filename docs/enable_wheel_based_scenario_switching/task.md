# タスク

## 依頼内容
ScenarioWorkspace において、従来の「縦スクロールで入力/結果を切替」機能を廃止した代替として、縦スクロールでシナリオ（左右の選択対象）を切り替えられるようにする。

## 必須条件
- 入力要素（`input` 等）にフォーカス中は切り替えを無効にする。
- 入力フォームや結果パネル内で通常スクロール可能な場合は、スクロール挙動を優先する。
- スクロール先がなくなったタイミングでのみシナリオ切替を実行する。

## 完了条件
- ホイール操作で前後シナリオへ遷移できる。
- 既存のボタンによる前後遷移は維持される。
- 型チェック、テスト、ビルドが通る。
