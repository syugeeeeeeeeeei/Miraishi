# 実装計画

## 1. 円形メニューの配置最適化

- メニュー半径・角度を見直し、すべての展開アイコンが右上側に出るよう調整する。
- ベース位置を中央から右寄せ寄りへ変更して、閉じたサイドパネルでも左端が欠けないようにする。

## 2. アイコンサイズと間隔調整

- 歯車アイコンと展開アイコンを、視認性を残しつつ小型化する。
- 半径を縮めてアイコン間距離を詰める。

## 3. 税金ルールAPIフォールバック強化

- `get-tax-schema` が未登録時は、内蔵デフォルト税スキーマでダイアログを開く。
- `update-tax-schema` が未登録時でも、セッション内オーバーライドとして再計算へ反映する。
- そのため、計算APIへ `taxSchemaOverride` を渡せるように main/preload/store を拡張する。

## 4. 検証

- `yarn typecheck`
- `yarn test --run src/main/lib/calculator.spec.ts`
