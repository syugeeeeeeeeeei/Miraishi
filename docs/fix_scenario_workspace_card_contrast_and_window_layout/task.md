# タスク

## 依頼内容
ScenarioWorkspace の 2 ペイン UI に対して、次の 2 点を修正する。

1. 入力項目カードが背景と同化してカード感が弱い。
2. 全画面では正常だが、ウィンドウ表示時にレイアウトが崩れる。

## 目的
- 入力フォームの情報ブロックを視覚的に判別しやすくする。
- 画面幅が狭いデスクトップウィンドウでも破綻しない表示を実現する。

## 完了条件
- 各入力セクションが背景から明確に分離されて見える。
- ScenarioWorkspace がウィンドウ幅に応じて崩れず表示される。
- TypeScript 型チェックとテスト、ビルドが通る。
