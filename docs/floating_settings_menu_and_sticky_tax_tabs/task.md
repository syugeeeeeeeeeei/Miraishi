# タスク

## 依頼内容
1. 設定メニューアイコンがサイドパネルの開閉に影響されて位置ずれするため、サイドパネルから独立したフローティングアイコンに変更する。
2. 税金ルールダイアログのモード切替タブが背景と同化して見づらいため、背景色と選択状態の視認性を改善する。
3. 税金ルールダイアログのモード切替タブをスクロール時にも操作しやすいよう sticky 化する。

## 完了条件
- 設定メニューがサイドパネル内部レイアウトから切り離され、パネル開閉時に位置が変わらない。
- 税金ルールダイアログのタブに明確な背景と選択状態スタイルがある。
- ダイアログ本文をスクロールしてもタブが上部に固定表示される。
