# 履歴ドキュメント（圧縮版）

このディレクトリは、過去の修正ドキュメントを「探しやすく読むため」に圧縮した履歴レイヤーです。
詳細原本は `../archive/legacy_docs_2026Q1/` に退避し、ここでは要点を参照します。

## 構成
- `index.md`: 全修正の索引（カテゴリ・要約・検証状況）
- `timeline.md`: 時系列の変更ログ
- `by_domain/*.md`: 領域別の統合サマリー

## 運用ルール
- 新規修正は、原則として個別3ファイルを増やさず `index.md` と該当 `by_domain` へ追記する。
- 大規模仕様策定で個別メモを作った場合も、実装完了後に本ディレクトリへ統合する。
- 原本を参照したいときは各エントリの `原本` リンクから `archive` を参照する。
