# 用語定義書

## 1. 本書の目的

本書は、Miraishi で使う用語を統一し、仕様解釈のぶれを防ぐための辞書である。  
追加機能が実装されても、既存用語の意味は変更しない。

## 2. プロダクト用語

| 用語 | 定義 |
|---|---|
| Miraishi（未来視） | 将来収入をシミュレーションし比較するデスクトップアプリ |
| シナリオ | 条件入力の1セット。比較の最小単位 |
| アクティブシナリオ | 現在表示・比較対象として選択されているシナリオ |
| 予測期間 | 将来収入を算出する年数 |
| 表示項目 | グラフに描画する指標（例: 額面年収、手取り年収） |

## 3. 収入・控除用語

| 用語 | 定義 |
|---|---|
| 基本給 | 毎月の基礎賃金 |
| 初任給（固定残業代込み） | 固定残業代を含む入社時点の月額給与。内部で基本給算出に利用される入力値 |
| 年間ボーナス | 1年で支給される賞与総額 |
| 給与成長率 | 年次で基本給へ乗算する成長率 |
| 手当 | 基本給や固定金額に基づいて加算される収入項目 |
| 固定手当 | 月額固定で加算される手当 |
| 割合手当 | 基本給等に対する割合で加算される手当 |
| 固定残業代 | あらかじめ定めた残業時間分の固定支給額 |
| 変動残業代 | 実残業時間条件に応じて変動する支給額 |
| 試用期間 | 一定期間のみ異なる賃金条件を適用する期間 |
| 額面年収 | 控除前の年間総収入 |
| 手取り年収 | 控除後に実際に受け取る年間収入 |
| 所得控除 | 課税所得算定時に差し引く控除額 |
| 社会保険料 | 健康保険、年金、雇用保険などの保険料 |
| 所得税 | 課税所得に税率表を適用して算出する税額 |
| 住民税 | 課税所得を基準に算出する地方税 |

## 4. システム用語

| 用語 | 定義 |
|---|---|
| メインプロセス | 永続化・計算・OS連携を担当する実行領域 |
| レンダラープロセス | UI描画とユーザー操作を担当する実行領域 |
| プリロード | 表示層へ安全なAPIを提供する中間層 |
| IPC | プロセス間通信。表示層と実行層の通信手段 |
| スキーマ | データ構造や検証ルールを定義した形式 |
| 税制スキーマ | 税率・保険料率・控除額を定義する外部データ |
| 税制スキーマV2 | `rules`・`formula`・`uiMeta` を持つ現行税制スキーマ構造 |
| 税務プロファイル（taxProfile） | 都道府県コードと業種コードを保持し、保険料率解決に使うシナリオ属性 |
| 文字式DSL | 計算式を文字列で表現する制限付き式言語。任意コード実行は不可 |
| スキーマスナップショット | 適用済み税制スキーマの履歴保存単位。復元・差分比較に利用される |
| uiMeta.items | スキーマ項目ごとの `name` / `description` / 関連計算式IDを定義する領域 |
| バリデーション | データが契約を満たすか検証する処理 |
| キャッシュ | 再計算を抑制するため一時保存した計算結果 |

## 5. UX用語

| 用語 | 定義 |
|---|---|
| コントロールパネル | シナリオ選択や管理操作を行う領域 |
| データビュー | 条件入力と結果確認を行う主表示領域 |
| グラフビュー | 比較可視化を行う補助表示領域 |
| ドロワー | 画面端から展開するオーバーレイUI |
| カルーセル表示 | 複数対象を順送りで表示する方式 |

## 6. 運用用語

| 用語 | 定義 |
|---|---|
| 後方互換 | 旧データ・旧操作を新実装でも成立させる性質 |
| コア契約 | 変更してはならない最小仕様セット |
| 拡張項目 | 既存契約を壊さず追加可能な仕様要素 |
| 単一正本（SSOT） | 真実の参照元を1か所に限定する運用方針 |
