# 取り扱い説明書（ユーザー向け）

## 1. この説明書について

本書は、Miraishi の日常利用に必要な基本操作をまとめたものです。  
追加機能が実装されても、ここで示す基本操作フローは継続して有効です。

## 2. できること

- キャリア条件をシナリオとして複数作成
- 将来の額面年収・手取り年収を予測
- シナリオ同士を表とグラフで比較
- 入力内容をローカル保存して継続利用

## 3. 画面の見方（概略）

- コントロールパネル: シナリオ作成、選択、検索、削除
- データビュー: 条件入力と計算結果確認
- グラフビュー: 複数シナリオの推移比較

## 4. 基本操作

### 4.1 シナリオを作成する

1. コントロールパネルで「新しいシナリオ」を選択
2. 追加されたシナリオを選択
3. タイトルをわかりやすい名前へ変更

### 4.2 条件を入力する

1. 給与・賞与・成長率を入力
2. 必要に応じて試用期間を有効化し条件を入力
3. 固定残業代の有無と条件を入力
4. 扶養情報を入力
5. 手当を追加し、金額と期間を設定

### 4.3 保存と再計算

- 入力後、フォーカスがカード外へ移動すると自動保存・再計算される。
- 再計算中は読み込み表示が出る場合がある。

### 4.4 結果を確認する

1. データビューで計算結果を確認
2. 年次行を選択し、収入/控除の内訳を確認

### 4.5 グラフで比較する

1. 比較したいシナリオを複数選択
2. 「グラフ表示」を開く
3. 予測期間、残業時間、表示項目を調整
4. 複数線の推移を比較

## 5. 効率よく使うコツ

- 比較テーマごとにシナリオ名を明確化する
- まず保守的条件、次に挑戦条件を作って差分を見る
- 予測期間を短期・中期・長期で切り替えて判断する

## 6. 注意事項

- 本アプリの結果は将来を保証するものではなく、意思決定支援の参考値。
- 税制や保険制度の改定により、現実の手取りは変動する可能性がある。
- 実際の契約条件（会社規定・就業規則）を優先して判断すること。

## 7. よくある質問（FAQ）

### Q1. シナリオを選択しても結果が出ない

- 入力値が極端でないか確認する。
- 一度別シナリオへ切り替え、再選択する。

### Q2. グラフボタンが押せない

- 計算対象となるシナリオが選択されているか確認する。

### Q3. 表示値が想定と違う

- 残業時間、手当期間、試用期間、扶養条件を再確認する。
- 額面と手取りの表示対象を取り違えていないか確認する。

## 8. トラブルシューティング

- アプリが起動しない: 再起動後も再現する場合、設定ファイルや配布パッケージ整合性を確認する。
- 計算エラー: 入力を段階的に戻し、どの項目変更で発生するか切り分ける。
- 動作が重い: 同時比較シナリオ数や予測期間を減らして確認する。

## 9. サポート時に伝えるとよい情報

- 利用OS
- 発生手順（どの入力後に起きたか）
- 対象シナリオ数
- 予測期間と残業時間設定
