# 修正内容の確認

## 1. フォント構成の見直し
- YAML エディターのフォントスタックを `ui-monospace` 優先に変更。
- ASCII のインデント幅が揃いやすい構成にして、空白幅の違和感を軽減。

## 2. スペース/タブ表示の改善
- `.cm-highlightSpace` を灰色背景から青ドット表示へ変更。
- `.cm-highlightTab` を灰色背景から青矢印表示へ変更。
- タブは `minWidth: 2ch` を設定し、狭く潰れて見える問題を抑制。

## 3. 行末空白の視認性
- `.cm-trailingSpace` の赤帯コントラストを強め、不要な空白を判別しやすくした。

## 4. 補助文言の更新
- エディター下部説明を、スペース/タブ/行末空白の可視化ルールに合わせて更新。

## 5. 検証
- `yarn typecheck:web`: 成功
- `yarn typecheck:node`: 成功
