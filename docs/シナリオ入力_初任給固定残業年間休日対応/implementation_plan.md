# 実装計画

## 1. 型とバリデーション拡張

- `Scenario` に `initialGrossSalary` と `annualHolidays` を追加する。
- 既存データ互換のため、`scenarioSchema` で不足項目を補完しつつ、入力値から `initialBasicSalary` を正規化する。

## 2. 計算ロジック更新

- 年間休日数から所定労働時間（月）を算出する。
- 初任給（固定残業代込み）・固定残業時間・年間休日数から初任基本給を逆算する。
- 固定残業代・変動残業代の時給計算で、固定の `160` ではなく算出した所定労働時間（月）を使う。

## 3. UI更新

- 入力フォームを新項目中心へ変更する（初任給・固定残業時間・年間休日数）。
- 算出された基本給・固定残業代（月額）をフォーム上に表示する。
- 計算フローの表示式を新ロジックに合わせて更新する。

## 4. テスト

- 既存テストを新仕様に追従させる。
- `yarn typecheck` と `yarn test --run` で回帰確認する。
